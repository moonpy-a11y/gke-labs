# GKE - Google Cloud Labs

This repository consolidates two core Google Kubernetes Engine (GKE) training repositories: `ak8s` and `training-data-analyst`. The goal is to provide a single, organized access point for GKE-focused laboratory content.

## Repository Contents

This consolidated repository contains the following sources:

### 1. `ak8s` (Architecting with GKE)
- **Description:** Content and labs focused on the architecture, deployment, and management of applications on GKE.
- **Topics:** Advanced networking, security, CI/CD, and multi-cluster management.
- **Location:** `ak8s/`

### 2. `training-data-analyst` (Data & Analytics)
- **Description:** Labs focusing on using GKE within the context of data analysis and machine learning workflows.
- **Topics:** Spark on GKE, Kubeflow, and data processing.
- **Location:** `training-data-analyst/`
